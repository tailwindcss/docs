## Usage

Use the `hue-rotate-{amount}` utilities alongside the `filter` utility to blur an element.

```html
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-indigo-400 rounded-md filter -hue-rotate-180"></div>
      <p class="relative font-extrabold text-white">-180</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-indigo-400 rounded-md filter -hue-rotate-60"></div>
      <p class="relative font-extrabold text-white">-60</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-indigo-400 rounded-md filter hue-rotate-0"></div>
      <p class="relative font-extrabold text-white">0</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-indigo-400 rounded-md filter hue-rotate-60"></div>
      <p class="relative font-extrabold text-white">60</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-indigo-400 rounded-md filter hue-rotate-180"></div>
      <p class="relative font-extrabold text-white">180</p>
    </div>
  </div>
</template>

<div class="filter **-hue-rotate-180** ..."></div>
<div class="filter **-hue-rotate-60** ..."></div>
<div class="filter **hue-rotate-0** ..."></div>
<div class="filter **hue-rotate-60** ..."></div>
<div class="filter **hue-rotate-180** ..."></div>
```

## Responsive

To control an element's hue rotation at a specific breakpoint, add a `{screen}:` prefix to any existing hue-rotate utility. For example, use `md:hue-rotate-60` to apply the `hue-rotate-60` utility at only medium screen sizes and above.

```html
<div class="filter hue-rotate-15 **md:hue-rotate-60** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.

## Customizing

You can customize which `hue-rotate` utilities are generated using the `hueRotate` key in the `theme` section of your `tailwind.config.js` file.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       hueRotate: {
+         '-270': '-270deg',
+         270: '270deg',
+       }
+     }
    }
  }
```

Learn more about customizing the default theme in the [theme customization documentation](/docs/theme#customizing-the-default-theme).
