## Usage

Use the `blur-{amount?}` utilities alongside the `filter` utility to blur an element.

```html fuchsia
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-fuchsia-400 rounded-md filter blur-0"></div>
      <p class="relative font-extrabold text-white">0</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-fuchsia-400 rounded-md filter blur-sm"></div>
      <p class="relative font-extrabold text-white">sm</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-fuchsia-400 rounded-md filter blur"></div>
      <p class="relative font-extrabold text-white">blur</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-fuchsia-400 rounded-md filter blur-lg"></div>
      <p class="relative font-extrabold text-white">lg</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-fuchsia-400 rounded-md filter blur-3xl"></div>
      <p class="relative font-extrabold text-fuchsia-900">3xl</p>
    </div>
  </div>
</template>

<div class="filter **blur-0** ..."></div>
<div class="filter **blur-sm** ..."></div>
<div class="filter **blur** ..."></div>
<div class="filter **blur-lg** ..."></div>
<div class="filter **blur-3xl** ..."></div>
```

## Responsive

To control an element's blur at a specific breakpoint, add a `{screen}:` prefix to any existing blur utility. For example, use `md:blur-lg` to apply the `blur-lg` utility at only medium screen sizes and above.

```html
<div class="filter blur-sm **md:blur-lg** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.

## Customizing

You can customize which `blur` utilities are generated using the `blur` key in the `theme` section of your `tailwind.config.js` file.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       blur: {
+         xs: '2px',
+       }
+     }
    }
  }
```

Learn more about customizing the default theme in the [theme customization documentation](/docs/theme#customizing-the-default-theme).
