## Usage

Use the `contrast-{amount?}` utilities alongside the `filter` utility to control an element's contrast.

```html amber
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-amber-400 rounded-md filter contrast-0"></div>
      <p class="relative font-extrabold text-white">0</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-amber-400 rounded-md filter contrast-50"></div>
      <p class="relative font-extrabold text-white">50</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-amber-400 rounded-md filter contrast-100"></div>
      <p class="relative font-extrabold text-amber-800">100</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-amber-400 rounded-md filter contrast-150"></div>
      <p class="relative font-extrabold text-amber-800">150</p>
    </div>
    <div class="relative h-16 flex items-center justify-center">
      <div aria-hidden="true" class="absolute inset-0 bg-amber-400 rounded-md filter contrast-200"></div>
      <p class="relative font-extrabold text-amber-800">200</p>
    </div>
  </div>
</template>

<div class="filter **contrast-0** ..."></div>
<div class="filter **contrast-50** ..."></div>
<div class="filter **contrast-100** ..."></div>
<div class="filter **contrast-150** ..."></div>
<div class="filter **contrast-200** ..."></div>
```

## Responsive

To control an element's contrast at a specific breakpoint, add a `{screen}:` prefix to any existing contrast utility. For example, use `md:contrast-150` to apply the `contrast-150` utility at only medium screen sizes and above.

```html
<div class="filter contrast-110 **md:contrast-150** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.

## Customizing

You can customize which `contrast` utilities are generated using the `contrast` key in the `theme` section of your `tailwind.config.js` file.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       contrast: {
+         25: '.25',
+       }
+     }
    }
  }
```

Learn more about customizing the default theme in the [theme customization documentation](/docs/theme#customizing-the-default-theme).
